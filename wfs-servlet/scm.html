Overview

This project uses Git to manage its source code. Instructions on Git use can be found at https://git-scm.com/documentation.

Web Browser Access

The following is a link to a browsable version of the source repository:

https://github.com/liturner/boscop-server/wfs-servlet

Anonymous Access

The source can be checked out anonymously from Git with this command (See https://git-scm.com/docs/git-clone):

$ git clone https://github.com/liturner/boscop-server.git

Developer Access

Only project developers can access the Git tree via this method (See https://git-scm.com/docs/git-clone).

$ git clone https://github.com/liturner/boscop-server.git

Access from Behind a Firewall
